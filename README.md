# MMDDesk v1.0 Beta

# 概要
C++/DirectX/DirectCompositionで開発している、デスクトップマスコット型MMDモデル(*.pmx)のビューワーです。VMD形式のモーション再生にも一応対応しています。

# 基本操作
- Ctrl+Alt+Gキーで移動や回転、拡大縮小UIを出せます
- Ctrl+Alt+Rキーでウインドウサイズ変更+移動UIを出せます（将来的には上記UIと統合予定）
- 上記UIでCtrl+Shift+スクロールでモデルの拡大縮小、左ドラッグで移動、右ドラッグで回転
- そのほか、設定やモーションの読み込みなどはタスクトレイから見ることができます

# ファイル・ディレクトリ構成
- `Models`ディレクトリ
    - デフォルトではこのディレクトリの`default.pmx`を読み込みます。
- `Motions`ディレクトリ
    - 再生したい`.vmd`モーションファイルをここに配置してください。
- `Shaders`ディレクトリ
    - 描画に使用するシェーダー(`.hlsl`/`.cso`)が格納されています。必要なら編集することもできます。シェーダーを変更した場合は、コンパイル済みシェーダー(`.cso`)を削除してください。
- `settings.ini`
    - 設定ファイルです。

# 動作環境
- **OS**: Windows 10 64ビット※
- **CPU**: AVX2対応CPU※
- **GPU**: DirectX 12対応GPU
<br>※手動ビルドで変更できます。32ビットでも実行可能です。

# 既知の不具合・制約
- モデルの一部が見切れる
  - つま先は仕様により見切れる場合があります。それ以外の場合、ウインドウを大きく拡大してください。
- ロードしたモデルが表示されない
  - 設定を開き、モデルのサイズを大きくすると表示されます
- 膝や足の挙動がおかしい
  - やや改善しましたが、それでも少し膝がカクっとなったりする可能性があります。
- 32ビットビルドで色がおかしい部分がある
