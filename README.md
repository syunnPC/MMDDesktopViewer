# MMDDesk v1.0 Beta

# 概要
Windows API/DirectComposition/DirectX 12で開発された、デスクトップマスコット型MMDモデル(*.pmx)のビューワーです。VMD形式のモーション再生にも一応対応しています。

# 基本操作
- モデルを左クリック+ドラッグで移動
- モデルを右クリック+ドラッグで回転
- Ctrl+Shift+スクロールでモデルの拡大縮小
- そのほか、設定やモーションの読み込みなどはタスクトレイから見ることができます

# ファイル・ディレクトリ構成
- `Models`ディレクトリ
    - デフォルトではこのディレクトリの`default.pmx`を読み込みます。
- `Motions`ディレクトリ
    - 再生したい`.vmd`モーションファイルをここに配置してください。
- `Shaders`ディレクトリ
    - 描画に使用するシェーダーが格納されています。必要なら編集することもできます。
- `settings.ini`
    - 設定ファイルです。

# 動作環境
- **OS**: Windows 10 64ビット※
- **CPU**: AVX2対応CPU※
- **GPU**: DirectX 12対応GPU
※手動ビルドで変更できます

# 既知の不具合・制約
- クリック透過が機能しない: モデルを大きくしすぎると操作に支障が出る可能性があります。
- マルチモニターで正しく描画されない場合がある: 接続されているGPUが異なるモニター間の移動で、モデルが描画されなくなる場合があります。
- モデルの一部が見切れる: モデルを大きく拡大してから縮小することで表示可能です
- モデルを回転させると大きくなる
- ロードしたモデルが表示されない: 設定を開き、モデルのサイズを大きくすると表示されます
